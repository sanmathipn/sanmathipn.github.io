Setup sync
Connect Stride to your Google Sheet so data syncs across all your devices automatically.
1
Create a Google Sheet
Go to sheets.new and name it Stride Data.
2
Add the script
Click Extensions → Apps Script. Delete existing code. Paste the full contents of google-apps-script.js (from your zip). Click Save.
3
Deploy as Web App
Click Deploy → New deployment → Web app. Execute as: Me. Access: Anyone. Click Deploy → Authorize → copy the URL.
